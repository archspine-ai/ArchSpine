# Tutorials

Learn ArchSpine by doing. Start here if you are new.

| Tutorial                          | Duration | What you will learn                           |
| --------------------------------- | -------- | --------------------------------------------- |
| [Quick Start](./quick-start)      | 5 min    | Install, scan, sync, and connect MCP          |
| [Your First Rules](./first-rules) | 10 min   | Write architecture rules and audit violations |

All tutorials use the ArchSpine repository itself as the example project.

## Prerequisites

- Node.js >= 20
- A Git repository (any language, any size)
- An LLM API key (optional for `scan --quick`; required for `sync`)
