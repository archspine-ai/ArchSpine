# ArchSpine 更新日志摘要

## 文档目的
本更新日志是 ArchSpine 项目所有公开变更的官方记录。它为开发者、用户和贡献者提供清晰、按时间顺序的新功能、改进和修复概览。通过阅读本文档，您可以快速了解版本间的变化，无需查阅每次提交日志或内部工单。

## 适用读者
- **开发者**：集成或扩展 ArchSpine 工作流的开发者。
- **用户**：需要了解特定版本提供哪些功能或修复的用户。
- **贡献者**：希望跟踪项目演进并确保工作与最新版本保持一致的贡献者。

## 本文档锚定的工作流与决策
- **核心 CLI 工作流**：`spine try`、`init`、`build`、`sync`、`publish`、`check`、`fix` 和 `mcp start` 在 1.0.0 版本中引入，构成所有 ArchSpine 操作的基础。
- **可靠性保证**（原子写入、基于锁的互斥、同步恢复）在每次发布中记录，指导升级决策和部署规划。
- **文档语言支持**：英文和简体中文文档的可用性在此追踪，帮助多语言用户找到合适的入口。
- 1.0.0 版本是首次开源发布，标志着项目对透明、社区驱动开发的承诺。

## 主要亮点（来自 1.0.0 版本）
- ArchSpine CLI 及本地 `.spine/` 控制平面的首次公开发布。
- 七个核心工作流及实验性功能（如 `spine mcp start`）。
- 内置可靠性：受保护写入、基于锁的互斥、同步恢复。
- 提供英文和简体中文的完整文档：包括 README、协议、运行手册、MCP、LLM 配置、成本管理及 God Mode 能力说明。
