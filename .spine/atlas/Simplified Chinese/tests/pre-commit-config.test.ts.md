# ArchSpine 预提交钩子配置测试套件

## 角色
该文件是一个 Vitest 单元测试套件，用于验证 ArchSpine 的预提交钩子配置及其解析逻辑。

## 主要职责
- 测试 `Config.getPreCommitResolution` 方法，确保返回正确的解析值及其来源。
- 验证 `Config.getHookSyncMode` 方法返回预期的同步模式。
- 为每个测试创建独立的临时目录，避免副作用。
- 每个测试结束后清理所有测试产物，保持严格的测试隔离。

## 重要不变项与负面范围
- **必须**是测试文件：导入 Vitest 并使用 `describe`/`it` 块。
- **必须**使用临时目录隔离测试环境。
- **必须**在每个测试后清理测试产物。
- **不包含**预提交钩子的生产运行时逻辑。
- **不直接**修改测试隔离之外的 git 钩子。
- **不进行**超出单元测试范围的配置模式验证。

## 导出/外部可见行为
该文件不导出任何生产接口，纯粹是一个测试套件。其外部可见行为仅限于通过或失败 Vitest 断言，以确认在不同环境和配置状态下预提交配置解析和钩子同步模式的正确性。
