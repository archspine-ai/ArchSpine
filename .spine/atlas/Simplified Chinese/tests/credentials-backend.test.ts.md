# ArchSpine – Windows DPAPI 凭据后端测试套件

## 角色
这是一个针对 Windows DPAPI 凭据后端的 Vitest 测试套件，用于验证密钥存储的安全性和正确性。

## 主要职责
- 确保密钥通过**标准输入**传递给 PowerShell，而非通过命令行参数，从而防止凭据通过进程检查泄露。
- 在隔离的模拟环境中测试凭据后端的**设置操作**。
- 在测试用例之间清理模拟和环境变量，保持测试隔离。

## 重要不变规则
- 测试文件必须使用 `.test.ts` 或 `.spec.ts` 后缀。

## 不涉及范围
- 非 Windows 凭据后端或跨平台凭据存储。
- 凭据检索（获取）或删除操作。
- 与更高级别凭据存储抽象的集成。

## 导出的外部可见行为
本测试套件不导出任何公共 API。它纯粹是一个测试工具，用于验证 Windows DPAPI 凭据后端的内部行为。
