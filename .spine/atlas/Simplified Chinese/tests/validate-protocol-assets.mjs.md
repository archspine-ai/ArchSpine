# ArchSpine 协议模式验证器

## 目的
本文档是一个 Node.js 验证脚本，用于确保 ArchSpine 系统中所有示例协议资产符合其定义的 JSON 模式。它充当协议完整性的守门人，在结构错误传播之前将其捕获。

## 上下文与受众
适用于 ArchSpine 协议的开发者和维护者，他们需要验证示例文件（包括 JSON 和带前置元数据的 Markdown）是否与预期的模式定义匹配。该脚本作为 CI 或本地开发工作流的一部分运行。

## 主要职责
- 从 schemas 目录加载 JSON 模式文件
- 验证示例 JSON 文件与其对应模式的匹配性
- 处理带前置元数据的 Markdown 规则文档的特殊验证
- 报告验证结果并以适当状态码退出

## 关键要点
- 自动从 schemas 目录加载所有 .schema.json 文件
- 针对预定义的示例文件集合，验证其与对应模式的匹配性
- 通过提取前置元数据和正文内容来处理 Markdown 规则文档
- 任何验证失败时以退出码 1 终止，适合集成到 CI 流水线中

## 不涉及范围
- 模式的生成或编写
- 协议规则的运行时强制执行
- 面向用户的文档或叙述性说明
