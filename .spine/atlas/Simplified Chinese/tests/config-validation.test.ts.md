# ArchSpine 配置系统 – 单元测试套件

本 Vitest 测试套件用于验证 ArchSpine 配置系统的配置加载、语言设置、钩子同步模式以及警告输出。

## 职责

- 为每个测试用例创建独立的临时目录（含 `.spine` 子目录），避免副作用。
- 验证语言配置加载及默认回退行为。
- 测试钩子同步模式的配置及其默认值。
- 通过 `console.warn` 间谍验证缺失或无效配置文件时的警告输出。

## 不涉及范围

- 超出临时目录的真实文件系统集成测试。
- 配置加载的性能或压力测试。
- 与语言或钩子同步模式无关的其他配置部分的验证。

## 不变约束

- 每个测试用例都会创建全新的临时目录，防止状态泄漏。
- 测试清理通过 `vi.restoreAllMocks()` 恢复所有模拟。
- 配置文件以 JSON 格式写入，并使用 `CURRENT_CONFIG_SCHEMA_VERSION`。

## 变更意图

架构意图是提供确定性、隔离的配置子系统单元测试，确保语言和钩子同步默认值正确，并在配置格式错误时发出警告。最近的变更意图标记为“gogogo”——可能是一次批量提交，未指明具体的配置验证变更，仅表示一般性进展。

## 公开接口

本测试套件不导出任何公开接口。
