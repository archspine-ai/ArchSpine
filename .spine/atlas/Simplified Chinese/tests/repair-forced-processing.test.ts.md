# ArchSpine 强制处理测试套件

本 Vitest 测试套件用于验证 ArchSpine 任务在修复策略条件下的强制处理行为。它确保 `ASTExtractionTask` 和 `SummarizationTask` 能够正确处理强制处理触发和状态转换。

## 主要职责

- 为每个测试用例设置独立的临时目录和模拟环境。
- 测试 `ASTExtractionTask` 和 `SummarizationTask` 在修复策略条件下的强制处理行为。
- 验证触发强制处理时的任务执行和状态转换。
- 在每个测试用例后清理所有测试工件并恢复模拟环境。

## 不涉及范围

- 生产环境中的任务执行或编排。
- 对文件系统的永久性修改。
- 与外部 API 或服务的集成。

## 不变约束

- 该文件是一个 Vitest 测试套件（`.test.ts`）。
- 每个测试使用独立的临时目录。
- 所有模拟环境在每个测试用例后恢复，工件被清理。

## 导出行为

该文件不导出任何公共 API。它是一个测试套件，作为 Vitest 测试运行器的一部分自动执行。
