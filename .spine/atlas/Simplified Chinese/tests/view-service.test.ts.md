# ViewService 集成测试套件

本文件是 ViewService 模块的 Vitest 集成测试套件。它通过模拟 LLM 客户端和文件系统夹具，在可控、可复现的环境中验证服务行为，确保从 Spine 单元生成视图的功能正确无误。

## 主要职责

- 创建临时测试目录和文件系统夹具，为 ViewService 测试提供隔离环境。
- 模拟 LLM 客户端响应，在不调用外部 API 的情况下模拟视图生成过程。
- 断言 ViewService 能正确处理 Spine 单元并生成预期的视图输出。
- 验证视图生成流程中的错误处理和边界情况。

## 关键不变性与负面范围

- **纯测试文件**：不得对生产系统产生任何副作用。
- **确定性模拟**：必须模拟所有外部依赖（LLM、文件系统），确保测试可重复。
- **资源清理**：每次测试后必须清理临时目录，防止资源泄漏。
- **不涉及范围**：本文件不处理生产部署、运行时服务托管、直接 LLM API 交互、测试会话外的工件持久化，以及用户界面渲染。

## 导出/外部可见行为

本文件不导出任何面向生产的 API。其唯一目的是在受控条件下运行集成测试，验证 ViewService 的内部逻辑。
