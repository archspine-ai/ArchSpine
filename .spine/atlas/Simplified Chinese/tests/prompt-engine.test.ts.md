# ArchSpine – 提示引擎基准测试套件

本文件是一个 **Vitest 基准测试套件**，用于评估提示引擎在大规模输入下的性能和扩展特性。它不是单元测试套件，而是专注于负载行为和生成产物的结构验证。

## 主要职责

- 使用任务特定预算，对包含大量内容和规则块的源提示产物进行压缩基准测试。
- 测试具有复杂导入/导出结构的源提示生成，并验证输出约束。
- 验证在扩展条件下，提示上下文产物构建与提示生成模块之间的集成。
- 断言生成的提示产物中的结构约束（行数、导入/导出数量）。

## 重要不变性

- 使用 Vitest 测试框架，采用 `describe`/`it`/`expect` 断言。
- 从 `src/infra/prompt-context.js` 和 `src/infra/prompt-rendering.js` 导入模块。
- 专注于在人为构造的大输入下的性能和扩展性。

## 不涉及范围

- 不测试单个提示上下文或提示渲染函数的单元行为。
- 不涉及与非提示引擎子系统的集成。
- 不进行端到端用户工作流验证。

## 导出/外部行为

本文件不导出任何公共 API。它是一个内部运行的测试套件，执行基准测试和断言。
