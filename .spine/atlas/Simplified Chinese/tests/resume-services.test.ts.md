# ArchSpine 可恢复聚合服务测试套件

## 角色
本套件是 Vitest 单元测试套件，用于验证 ArchSpine 系统中可恢复聚合服务的行为，确保中断的同步任务能够在服务边界之间正确恢复。

## 主要职责
- 模拟 `SyncService`、`CheckService` 和 `FixService`，将恢复逻辑测试与生产依赖隔离。
- 验证中断的同步候选者能否正确反馈到扫描和强制同步路径中。
- 使用 Vitest 的 `vi` 模拟工具（`vi.mock`、`vi.resetModules`）在测试用例之间重置状态。
- 强化扫描恢复验证和回退路径逻辑，反映近期对鲁棒性的改进。

## 关键不变性与排除范围
- **不变性：** 严格遵循 Vitest 测试框架约定。所有测试均通过模拟隔离单元行为，不依赖真实服务实现。
- **排除范围：** 本套件不测试生产服务实现、直接 HTTP 或数据库交互，以及 UI/CLI 渲染。它专注于同步、扫描和修复操作中的恢复逻辑。

## 导出行为
该测试套件不导出任何供外部使用的公共接口。其主要价值在于内部验证可恢复聚合逻辑，确保同步任务中断时状态能够正确持久化和恢复。
