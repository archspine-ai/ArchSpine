# SpineDB 批量提交测试套件

本 Vitest 测试套件为 SpineDB 的批量提交操作和事务完整性提供自动化回归覆盖。它确保批量提交能够原子性地执行，并在发生错误时正确恢复。

## 主要职责

- **SQLite 模拟**：模拟 SQLite 的 `prepare` 方法，以拦截并验证批量提交的 SQL 语句。
- **回滚验证**：验证批量提交在遇到约束违反时能够完全回滚。
- **测试环境管理**：在每个测试前后创建并清理临时测试目录。
- **原子性断言**：确认批量操作保持原子性，并且错误处理按预期工作。

## 不涉及范围

- 对批量提交上下文之外的单个 SpineDB 方法进行单元测试。
- 使用真实数据库实例进行集成测试。
- 批量操作的性能或负载测试。

## 重要不变规则

- 测试文件必须以 `.test.ts` 或 `.spec.ts` 结尾（规则：`test-file-suffix`）。

## 变更意图

架构意图是为 SpineDB 批量提交的原子性和错误恢复提供自动化回归覆盖。此文件标志着 v1.0.0 版本的最终关闭；最近的变更集中未检测到对此测试文件的具体修改。

## 公开接口

此测试套件不导出任何公开 API。
