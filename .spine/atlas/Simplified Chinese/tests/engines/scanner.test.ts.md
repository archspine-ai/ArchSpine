# ArchSpine Scanner 集成冒烟测试

本文件是一个 Vitest 集成测试套件，用于在受控的临时环境中验证 Scanner 引擎的基本文件发现和策略应用行为。

## 主要职责

- 创建并管理临时目录，用于隔离的测试执行。
- 生成模拟扫描目标的合成文件系统夹具（目录和文件）。
- 使用已解析的扫描策略调用 Scanner 引擎执行文件发现。
- 断言 Scanner 正确包含预期文件并排除被忽略的模式。

## 不包含的范围

- 对单个 Scanner 方法的单元测试。
- Scanner 引擎的性能或压力测试。
- 测试 Scanner 之外的其他引擎或核心组件。

## 重要不变规则

- 测试文件必须以 `.test.ts` 或 `.spec.ts` 结尾（规则：test-file-suffix）。

## 导出的或外部可见的行为

本文件不导出任何公共 API 接口。它是一个在 Vitest 框架下运行的测试套件，不供外部使用。
