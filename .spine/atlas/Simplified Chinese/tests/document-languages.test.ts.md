# ArchSpine – 文档语言选择测试套件

本 Vitest 单元测试套件用于验证 ArchSpine 镜像系统中文档语言选择与排序的核心逻辑。它确保语言选项以一致且明确的顺序呈现，并且高容量语言分隔符的行为符合预期。

## 主要职责

- **语言排序验证：** 确认 `getDocumentLanguageChoices` 首先返回稳定语言，接着是一个禁用的分隔符，最后是高容量语言。
- **分隔符完整性：** 断言高容量语言分隔符的正确索引和属性。
- **默认值检查：** 验证默认选中的语言值是否与预期默认值一致。
- **常量验证：** 确保 `HIGH_CAPACITY_LANGUAGE_SEPARATOR` 常量包含预期的描述性说明。

## 重要不变项与排除范围

- **不变项：** 测试文件后缀必须为 `.test.ts` 或 `.spec.ts`（已符合）。
- **排除范围：** 本套件**不**涵盖：
  - 非语言选择的 CLI 功能
  - 与外部服务的集成测试
  - UI 渲染或用户交互

## 导出行为

该测试套件不导出任何公共接口；它纯粹是一个内部验证工具。其主要外部可见行为是对正确语言排序和分隔符位置的断言，这直接影响面向用户的语言选择界面。
