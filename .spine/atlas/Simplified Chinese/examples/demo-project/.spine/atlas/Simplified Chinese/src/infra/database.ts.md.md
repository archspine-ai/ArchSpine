# ArchSpine – 数据库基础设施模块

## 目的
本文档定义了数据库基础设施模块，负责管理底层连接状态，并提供通用的查询接口用于执行类似 SQL 的命令。它处理数据库连接的完整生命周期，并作为应用程序数据持久化的具体实现层。

## 上下文与受众
本文档面向需要理解或修改数据库访问层的开发者和维护者。适用于任何从事数据持久化、连接管理或基础设施级数据库操作的人员。

## 主要职责
- 实现 `Database` 类以管理底层连接状态
- 提供通用的 `query` 方法用于执行类似 SQL 的命令
- 处理数据库连接的连接/断开生命周期
- 作为应用程序数据持久化的具体实现层

## 不包含的范围
- 高级 ORM 或查询构建器抽象
- 数据库迁移或模式管理
- 连接池或高级性能调优

## 关键要点
- `Database` 类是管理连接状态的核心组件
- 提供了通用的 `query` 方法用于执行类似 SQL 的命令
- 模块处理连接/断开的生命周期事件
- 这是数据持久化的具体实现层
