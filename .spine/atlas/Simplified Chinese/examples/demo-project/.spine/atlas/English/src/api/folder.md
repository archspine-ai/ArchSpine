`api` 目录是 ArchSpine 镜像系统的对外接口层。它作为所有外部交互的入口，负责将传入数据转换为领域友好的结构，并处理请求验证以及向调用方传播错误信息。该层通过 `CreateUserHandler` 模块编排用户创建请求。

该目录包含两个主要子项：`folder.md`（描述公共 API 与接口层职责的文档文件）和 `handler.ts.md`（作为系统入口的核心 API 请求处理器）。最重要的实现领域包括请求验证管道、数据格式转换以及 `CreateUserHandler` 的编排逻辑。
