`examples/demo-project/src` 目录是演示项目的后端应用源代码，按照 API、领域和基础设施三层结构组织。主要子目录包括：

- **`api`**：用户创建请求的 HTTP 入口，负责处理传入请求并将任务委派给领域层，同时管理数据库连接。
- **`domain`**：包含用户实体的业务逻辑，提供创建和检索功能，并使用内存存储实现。
- **`infra`**：数据库连接管理的桩模块，提供最小化的 `Database` 类，用来模拟 SQLite 或 PostgreSQL 的连接设置与状态追踪。

最重要的实现领域是用户创建与检索，这三个层相互配合：`api` 层提供端点，`domain` 层封装业务规则，`infra` 层处理数据库连接。这种清晰的分层结构使得系统易于扩展或替换组件。
