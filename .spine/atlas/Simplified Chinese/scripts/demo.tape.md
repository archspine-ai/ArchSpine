## 目的

本文档是一个终端录制脚本，用于生成展示 ArchSpine CLI 典型工作流程的演示 GIF。它的存在是为了让读者在不安装或配置工具的前提下，快速直观地了解命令行界面的使用方式。

## 受众

本脚本适用于希望直观了解如何使用 ArchSpine 核心命令（`sync`、`check`、`fix`）的开发者及用户。演示在模拟提供者（mock provider）环境下进行，读者只需基本终端操作知识，无需深入了解 ArchSpine 内部实现。

## 关键工作流程与决策

- **先构建**：演示从运行 `npm run build` 开始，确保 CLI 已准备就绪。
- **清理状态**：同步前，脚本明确删除 `.spine` 缓存文件（`index`、`atlas`、`cache.db`、`manifest.json`、`languages.json`、`.lock`）。这一决策使工作流程建立在干净、可重现的起点上。
- **使用模拟提供者**：通过环境变量 `SPINE_PROVIDER=mock` 隔离测试环境，避免依赖真实存储后端，使演示自包含。
- **命令顺序**：清理缓存后依次执行 `sync`、`check`、`fix`（fix 时输入 `y` 确认）。这个顺序体现了 ArchSpine 的典型工作流：先同步数据，再校验，最后自动修复问题。

## 本文档的重要性

此录制脚本锚定了 ArchSpine 的演示与新手上手流程。它精确定义了演示 GIF 中用户将看到的步骤，确保文档展示与实际演示内容一致。同时，它还为开发人员提供了一个可复现的测试场景，用于验证 CLI 的行为。
