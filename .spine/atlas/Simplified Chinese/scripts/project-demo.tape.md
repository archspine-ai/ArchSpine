# ArchSpine CLI 演示脚本摘要

## 目的
该文档是一个终端录制脚本（VHS 磁带），用于直观展示 ArchSpine 项目的核心构建与诊断工作流。其存在意义是让新用户或评估者无需阅读长篇文档，即可快速了解在实战中构建和运行 ArchSpine 是怎样的体验。

## 目标读者
主要面向开发者或技术评估人员，他们希望在阅读书面文档之前，通过一段实时的终端演示来了解 ArchSpine。脚本假定读者熟悉 Node.js 构建流程和命令行工具。

## 锚定的决策与工作流
- **构建验证**：脚本确认 ArchSpine 可以通过 `npm run build` 成功从源代码构建。这锚定了“仓库已准备好进行开发或评估”的决策。
- **CLI 可用性**：通过依次运行 `sync --fast`、`info` 和 `scan --dry-run`，脚本验证了 CLI 的基本功能正常，项目状态查看和扫描命令可按预期工作。
- **新手引导**：演示展示了克隆仓库后的典型第一步，锚定了用户的后续操作——同步项目状态、检查配置、执行干运行扫描，且无需依赖任何外部服务。

## 演示的工作流
1. **从源代码构建** — `npm run build`
2. **快速同步** — `node dist/cli/index.js sync --fast`
3. **查看项目信息** — `node dist/cli/index.js info`
4. **干运行扫描** — `node dist/cli/index.js scan --dry-run`

## 关键要点
- ArchSpine 可零外部服务依赖，仅需最小配置即可构建和运行。
- CLI 提供三个核心命令，覆盖了“构建-同步-检查-扫描”的典型流程。
- 该演示完全自包含，使用标准终端环境（zsh、Catppuccin Mocha 主题，可配置打字速度和窗口尺寸）。
