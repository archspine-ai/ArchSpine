# ArchSpine 基准测试运行器

本文档实现了一个实时基准测试运行器，用于测试 ArchSpine 系统基于 AST 的语义提取能力，以及通过 LLM 驱动的摘要生成和验证功能，覆盖多种代码场景。

## 上下文与目标读者

面向 ArchSpine 系统的开发者和维护者，用于验证语义提取管道的准确性和鲁棒性。基准测试覆盖小文件、大文件、多导入模块、规则密集型验证以及易漂移场景等边界情况。

## 主要职责

- 定义并执行针对 AST 提取和 LLM 驱动的摘要生成/验证的基准测试场景
- 提供涵盖小文件、大文件、多导入、规则密集和易漂移场景的示例配置
- 集成 OpenAI 兼容的 LLM 提供商进行语义分析
- 记录基准测试结果并生成摘要报告

## 不包含的内容

- ArchSpine 系统的生产部署或运行时执行
- 面向用户的文档或最终用户指南
- AST 提取内部机制的详细算法描述

## 关键要点

- 基准测试覆盖 7 个不同场景，包含摘要和验证两种任务模式
- 集成 OpenAI 兼容的 LLM 提供商进行语义分析
- 包含依赖上下文和规则数据，用于测试约束感知的验证能力
- 结果记录到 `externaldocs/logs/` 目录，便于分析和回归跟踪
