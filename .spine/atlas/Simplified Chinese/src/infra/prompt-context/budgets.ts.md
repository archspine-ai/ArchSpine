# ArchSpine – `calculateSourcePromptBudgets` 工具函数

## 角色
这是一个纯基础设施工具函数，用于根据任务模式、验证策略和源代码工件计算提示令牌预算。它无副作用，输出完全由输入参数决定。

## 主要职责
- 根据给定的任务模式和输入覆盖项，确定适用的提示层级和验证策略。
- 基于解析后的任务模式、验证策略和层级，从常量中选择正确的预算配置文件。
- 从提供的源代码内容和骨架中计算行数和导入数。
- 应用配置文件中的乘数和钳位值，计算出最终的预算分配（如 `maxTokens`、`reservedTokens`）。

## 关键不变性与排除范围
- **纯函数**：无副作用，无 I/O，无状态变更。
- **确定性**：输出仅依赖于输入参数。
- **排除范围**：本工具不负责编排任务、管理 API 调用、持久化或缓存预算结果，也不提供面向用户的 CLI 或 UI。

## 公开接口
- **`calculateSourcePromptBudgets`** – 唯一导出的函数。调用方传入任务模式、验证策略、源代码内容和骨架，即可获得完整的预算对象。
