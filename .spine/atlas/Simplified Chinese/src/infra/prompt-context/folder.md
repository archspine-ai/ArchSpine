`src/infra/prompt-context` 目录是 ArchSpine 提示策略编排系统的核心神经中枢。它负责提示生成的完整生命周期：根据任务模式解析策略、计算令牌预算、修剪多余内容、生成诊断信息，并以类型安全的方式解析配置。值得注意的子模块包括：`artifacts.ts` 中的编排服务，用于协调上述所有步骤；`budgets.ts` 中的令牌预算计算器；`constants.ts` 中的类型化常量定义；`diagnostics.ts` 中的诊断分析器；`policy.ts` 中的策略解析器；`trim.ts` 中的文本修剪工具；`parsers.ts` 中的配置解析器；以及 `types.ts` 中的全面类型定义。最具影响力的实现领域包括：策略解析（如何根据模式选择默认和验证策略）、预算计算（确保提示大小在限制范围内）以及内容修剪（通过规则块过滤优化相关性）。这些模块共同确保为 LLM 编排系统提供一致、高效且遵循规则的提示组装。
