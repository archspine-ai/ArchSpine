# ArchSpine – 漂移事件数据访问

## 角色
用于在 SQLite 中持久化和检索语义漂移审计事件的基础设施数据访问函数。

## 主要职责
- 将漂移事件记录插入 SQLite 数据库，包含文件路径、时间戳、先前角色、职责和原因。
- 查询给定文件路径的最新漂移事件，将存储的 JSON 职责解析回数组。

## 重要不变性与否定范围
- 暴露用于漂移事件持久化和检索的低级 SQLite 操作。
- 依赖预编译语句和 JSON 序列化来存储职责数组。
- **不**编排更高级别的审计工作流或业务逻辑。
- **不**提供用于漂移事件管理的公共 API 外观；调用者直接与数据库语句交互。

## 最重要的导出行为
- `recordDriftEvent` 函数（根据使用模式和先前契约推断）
- 用于检索漂移事件的查询函数（根据使用模式推断）
