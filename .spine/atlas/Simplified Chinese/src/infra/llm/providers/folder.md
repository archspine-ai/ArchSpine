`src/infra/llm/providers` 目录包含了实现具体大语言模型（LLM）提供者以及相关工具模块的代码，这些模块使 ArchSpine 的镜像系统能够生成语义摘要。该目录可分为三部分：三个具体的提供者客户端和一个共享的工具模块。  

提供者实现包括 `gemini.ts`（封装 Google Gemini API 以生成结构化的摘要）、`openai.ts`（兼容 OpenAI 的客户端，同时负责编排提示词的组装，用于语义分析流水线）以及 `mock.ts`（测试用的模拟客户端，用于单元测试和集成测试中模拟 LLM 的响应）。  

`utils.ts` 模块提供共享的基础设施：它将内容组装为提示词、解析结构化响应，并合并不同提供者的使用数据。这些文件共同构成了 ArchSpine 为多种文件类型生成语义摘要的核心机制。
