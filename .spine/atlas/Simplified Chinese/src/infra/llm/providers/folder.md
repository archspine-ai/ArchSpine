providers 目录位于 src/infra/llm 下，包含用于生成 ArchSpine 镜像系统结构化语义摘要的具体 LLM 提供程序客户端。该目录包含三个提供程序模块：gemini.ts（Google Gemini API）、openai.ts（兼容 OpenAI 的 API）以及 mock.ts（用于单元测试和集成测试的测试替身）。共享工具模块 utils.ts 负责内容组装、结构化响应解析以及使用量聚合。值得注意的是，OpenAI 客户端直接嵌入了提示词生成和解析逻辑，这违反了基础设施外观模式设计原则。该目录是 LLM 摘要生成管线的操作核心，将各提供程序的特定 API 调用转换为一致的镜像系统结构化输出。
