# PromptBuilder — 提示组装工具类

## 角色
`PromptBuilder` 是一个流式构建器类，负责编排结构化提示块的组装，用于 AI 代理交互。它提供清晰、可链式调用的接口，通过依次添加提示部分来构建提示。

## 主要职责
- **流式构建器接口**：提供如 `setIdentity(role, target)` 等方法，用于顺序添加提示部分。
- **委托渲染**：依赖从专用资产目录导入的模板函数来渲染各个块（如身份、指令、上下文）。
- **内部聚合**：将渲染后的块收集到内部字符串数组中，然后拼接成最终提示字符串。
- **公共 API**：导出 `PromptBuilder` 类作为提示构建的主要入口点。

## 重要不变性
- 导入严格限制为来自 `assets` 目录的模板渲染函数。
- 所有构建器方法返回 `this` 以支持方法链式调用。
- 内部状态是一个简单的字符串数组，按添加顺序组装。

## 不涉及范围
- **不**直接渲染或格式化原始文本（此职责委托给模板函数）。
- **不**管理 AI 模型交互或 API 调用。
- **不**编排核心基础设施服务或运行时任务。

## 导出的行为
- **类**：`PromptBuilder`
- **方法**：`PromptBuilder.setIdentity(role: string, target: string): this` — 向提示中添加身份块。
