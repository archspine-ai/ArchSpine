# ArchSpine – 视图生成评分引擎

## 角色
该模块是一个评分引擎，根据源文件成为公共 API 表面的可能性对其进行排序。它在视图生成期间用于识别哪些文件应作为公共接口暴露。

## 主要职责
- 应用多因素评分算法对文件进行分类和排序，以确定公共表面候选。
- 根据可配置阈值过滤掉被抑制的路径和低分候选。
- 基于语义公共表面声明、导出计数、内部消费者和重新导出放大计算分数。
- 对特定表面类型（CLI、MCP、配置、路由）应用加分，并对仅类型文件应用扣分。
- 将排序输出限制为最大项目数（`MAX_PUBLIC_SURFACE_ITEMS`）。

## 重要不变性
- 在加载的 `SpineUnit` 数据结构上运行。
- 输出是排序和过滤后的 `ScoredCandidate` 对象列表。
- 评分基于可配置因子和阈值具有确定性。

## 不涉及范围
- 直接 HTTP 或网络通信
- 用户界面渲染
- 持久化或数据库操作
- 运行时服务编排（它是一个纯评分引擎）

## 导出的行为
主要导出的行为是一个函数或类，它接受 `SpineUnit` 对象集合和配置参数，并返回一个排序后的 `ScoredCandidate` 对象列表，代表最可能的公共 API 表面文件。
