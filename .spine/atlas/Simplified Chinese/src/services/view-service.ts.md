# ViewService — 架构视图编排器

## 角色
从已索引的代码库数据中生成并管理架构视图的服务编排器。

## 主要职责
- 通过专用模块编排架构图、公共表面和风险热点视图的推导。
- 使用 ViewRenderer 将推导出的视图数据渲染为 Markdown 和 JSON 制品。
- 通过 OutputManager 将视图制品持久化到文件系统。
- 当视图定义被移除时，删除过时的视图制品。

## 不包含的范围
- 直接扫描或索引源代码（委托给 ViewIndexLoader）。
- 定义视图逻辑或渲染模板（委托给视图专用模块）。
- 底层文件 I/O（委托给 OutputManager）。

## 不变约束
- 依赖基础设施模块（OutputManager）进行输出操作。
- 依赖 `src/services/view/` 下的视图专用模块进行视图推导和渲染。
- 服务运行时边界：编排视图生成，但不包含视图专用逻辑。

## 导出接口
- `ViewService`（导出类）

## 架构意图
促进视图推导逻辑与编排之间的清晰分离，支持模块化视图添加和制品管理。
