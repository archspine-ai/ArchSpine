# ArchSpine – 运行时编排服务

## 角色
基础设施编排服务，用于管理可恢复的命令执行会话，支持检查点验证和受保护输出的变异检测。

## 主要职责
- 编排运行时命令会话（同步、检查、修复）并管理其生命周期。
- 验证并加载执行检查点，以支持可恢复操作。
- 通过 spine-gate 集成检测并警告受保护输出的变异。
- 在任务执行期间管理锁的获取与释放，防止冲突。
- 验证运行时配置和检查点状态。

## 不涉及范围
- 直接的视图渲染或 UI 逻辑。
- 底层文件 I/O 或数据库操作（委托给基础设施模块）。
- 特定任务的业务逻辑（委托给任务运行时）。

## 不变约束
- 依赖基础设施模块进行检查点存储、变异检测、I/O 和写入访问控制。
- 导出 `RuntimeCommand` 类型用于命令规范。
- 编排会话生命周期，但不实现核心任务逻辑。

## 公开接口
- `RuntimeCommand` 类型
- `RuntimeSessionOptions` 接口

## 变更意图
- **架构意图：** 提供一个可恢复、安全的编排层，用于命令执行，支持检查点验证和变异保护。
- **近期变更意图：** 验证运行时配置和检查点状态。
