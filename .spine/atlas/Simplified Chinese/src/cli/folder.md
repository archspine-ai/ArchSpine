`src/cli` 目录是 ArchSpine 系统的入口点和命令行界面层。它负责处理用户命令、解析参数、渲染帮助信息以及执行运行时引导。内容逻辑上分为工具模块和三个子文件夹：`commands`、`init` 和 `repo`。关键源文件包括 `index.ts`（主路由器及入口点）、`cli-utils.ts`（用于语言发现结果和横幅格式化的 UI 工具）、`document-languages.ts`（多语言文档层的配置类型）以及 `help.ts`（帮助文本渲染器）。`commands` 子文件夹包含具体的命令适配器，负责参数解析并委托给核心服务。`init` 子文件夹存放项目初始化与引导子系统。`repo` 子文件夹提供用于管理仓库产物策略的 CLI 适配器。这些组件共同构成了完整的命令行接口，用户通过它与 ArchSpine 的语义镜像工作流进行交互。
