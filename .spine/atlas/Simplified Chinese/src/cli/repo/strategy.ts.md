# ArchSpine CLI 构件策略适配器

## 角色
ArchSpine 系统中用于仓库构件策略管理的 CLI 命令适配器。

## 主要职责
- 解析并验证来自 CLI 参数的构件策略输入。
- 通过委托给仓库管理服务来协调仓库策略的检查与应用。
- 为策略操作提供面向用户的控制台输出，包括警告和分步反馈。
- 与系统配置集成，以获取持久化和初始化的构件策略。

## 重要不变项与负面范围
- 必须保持为薄命令适配器，将核心逻辑委托给服务。
- 不得包含仓库策略评估或应用的业务逻辑。
- 必须为 CLI 用户提供清晰的控制台反馈。
- 范围外：实现仓库策略逻辑（由仓库管理服务处理）、持久化配置更改（由配置基础设施处理）以及构件管理的直接文件系统操作。

## 最重要的导出行为
- `parseArtifactStrategy(value: string | undefined): ArtifactStrategy | undefined` — 解析并验证来自 CLI 参数的构件策略输入，如果输入缺失或无效则返回 undefined。
