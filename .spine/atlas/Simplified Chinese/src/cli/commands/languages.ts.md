# ArchSpine – 语言命令适配器

## 角色
该模块是一个轻量级的 CLI 命令适配器，用于 ArchSpine 系统中交互式文档语言配置管理。它负责连接用户界面的 CLI 命令与后端服务，自身不包含业务逻辑。

## 主要职责
- 提供交互式提示，让用户从可用选项中选择文档语言。
- 从系统配置中列出当前已配置的语言。
- 从清单中加载并显示语言快照，供用户参考。
- 通过基础设施层将更新后的语言选择保存回配置。

## 关键不变项与负面范围
- **必须保持为薄命令适配器** – 所有业务逻辑和持久化操作均委托给服务和基础设施模块。
- **不得包含** 管道、扫描或数据转换逻辑。
- **不涉及范围：** 核心语言发现或验证逻辑（由 `document-languages` 模块处理）、直接持久化操作（由 Config 服务处理），以及语言处理或分析的业务逻辑。

## 最重要的导出/外部可见行为
- 公开 `ExecuteLanguagesCommandOptions` 接口，该接口定义了执行语言命令的契约。
