# ArchSpine – 上帝模式 CLI 处理器

## 角色
该模块作为上帝模式功能的 CLI 命令处理器，负责在将实际工作委托给上帝引擎之前，向用户展示警告并获取确认。

## 主要职责
- 通过 `ExecuteGodCommandOptions` 接口定义上帝模式命令的命令行选项。
- 展示关于上帝模式非生产性质及覆盖行为的警告。
- 检查 `--force` 标志以跳过交互式确认。
- 在继续执行前提示用户明确确认（除非强制）。
- 将执行委托给上帝引擎的 `runGodMode` 函数。

## 重要不变性与负面范围
- **必须保持薄适配器** – 该模块不得吸收核心逻辑，仅处理 CLI 路由和用户交互。
- **无管道或持久化逻辑** – 不实现文档生成，也不持久化或读取生成的文档。
- **无复杂编排** – 不管理多步骤管道。

## 最重要的导出/外部可见行为
- `ExecuteGodCommandOptions` 接口 – 定义上帝模式命令的 CLI 选项。
- `executeGodCommand` 函数 – 主入口点，协调警告、确认和委托给引擎。
