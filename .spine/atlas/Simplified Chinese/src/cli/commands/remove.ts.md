# ArchSpine 移除命令适配器

## 角色
作为 `remove` 操作的 CLI 命令适配器，负责协调从仓库中清理 ArchSpine 管理的工件。

## 主要职责
- 解析命令行参数以确定移除范围和确认要求。
- 根据配置状态协调移除 ArchSpine 管理的文件（代理指令、git 属性、git 忽略、搜索忽略、spine 忽略）。
- 管理从仓库中卸载 ArchSpine 管理的 Git 钩子。
- 在执行破坏性移除操作前提供用户确认提示。

## 重要不变项与负面范围
- **必须保持为薄命令适配器**，将所有文件和钩子操作委托给专用工具模块。
- **不得包含确定 ArchSpine 管理工件构成的业务逻辑**；该逻辑属于配置服务。
- 不在范围内：实现底层文件系统移除逻辑、管理配置状态的持久化、执行核心流水线或扫描逻辑。

## 最重要的导出行为
- 将 `ExecuteRemoveCommandOptions` 接口作为其主要公共表面暴露。
