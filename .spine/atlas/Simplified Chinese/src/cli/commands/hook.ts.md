# ArchSpine – 钩子命令适配器

## 角色
ArchSpine 系统中用于管理 Git 钩子集成与同步的 CLI 命令适配器。

## 主要职责
- 从字符串输入解析钩子同步模式，确保配置有效。
- 根据用户命令参数，通过配置更新启用或禁用预提交 Git 钩子。
- 当钩子被触发时，委托给专用同步模块执行同步工作流。
- 向用户提供当前钩子配置和同步模式的状态反馈。

## 不涉及范围
- 实现核心同步管道逻辑（委托给 sync.js）。
- 管理 Git 钩子脚本内容（委托给 git-hook.js 工具）。
- 持久化配置状态（由 Config 服务处理）。

## 不变约束
- 必须保持为薄命令适配器；所有管道逻辑必须委托给服务/引擎。
- 不得包含扫描、分析或修复的业务逻辑。

## 公开接口
- `ExecuteHookCommandOptions` 接口
- `parseHookSyncMode` 函数
