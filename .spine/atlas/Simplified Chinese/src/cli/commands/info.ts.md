# ArchSpine – `info` 命令处理程序

## 角色
该模块是 `info` 命令的 CLI 处理程序，作为一个**薄适配器**，负责编排信息报告引擎的执行，不包含任何业务逻辑。

## 主要职责
- 定义 `ExecuteInfoCommandOptions` 接口，指定运行时选项，如根目录和命令参数。
- 调用 `runInfoReport` 引擎，传入从命令行参数和默认运行时 I/O 推导出的配置。
- 提供错误处理，捕获内部错误并将其映射为结构化的 `ArchSpine` 错误，确保 CLI 输出的一致性。

## 重要不变性与负面范围
- **必须保持为薄 CLI 适配器** – 所有管道逻辑都委托给引擎和服务。
- **不得**执行实际的信息报告逻辑或数据聚合。
- **不得**直接访问持久化或数据库层。
- **不得**实现超出命令编排范围的任何业务逻辑。

## 公开接口
- `ExecuteInfoCommandOptions`（接口）
- `executeInfoCommand`（异步函数）

该模块是 `info` 命令的干净入口点，确保了关注点分离和可维护性。
