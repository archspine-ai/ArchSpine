# ArchSpine 引导命令适配器

## 角色
该模块是一个 CLI 命令适配器，负责在目标仓库中引导 ArchSpine 配置及其相关工件。它是使用 ArchSpine 的托管文件、脚本、钩子和规则模板初始化仓库的入口点。

## 主要职责
- 提示用户选择或确认用于初始化的工件策略（例如 `index`、`atlas` 或 `hybrid`）。
- 通过仓库管理服务协调托管仓库文件（例如 `.gitattributes`、`.gitignore`）的同步。
- 协调将包脚本注入仓库并安装推荐的规则模板。
- 管理预提交完整性检查的 git 钩子的安装和配置。

## 重要不变项与负面范围
- **必须保持为薄命令适配器** — 它将所有核心操作委托给专用服务，不得包含仓库管理或工件生成的业务逻辑。
- **不在范围内**：实现管道或持久化逻辑、执行超出协调范围的直接文件系统操作，或实现工件生成的业务逻辑。

## 最重要的导出行为
- **`runRepositoryBootstrap`** — 编排整个引导过程的主要公共函数。这是该模块唯一对外可见的表面。
