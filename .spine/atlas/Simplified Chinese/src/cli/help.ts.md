# ArchSpine CLI 帮助文本渲染器

## 角色
该模块是 **ArchSpine 命令行界面的帮助文本渲染器**，负责以可读、格式化的方式向用户呈现帮助信息。

## 主要职责
- **通用帮助显示**：格式化并打印主帮助信息，列出所有可用命令并按类别分组（例如：设置与配置、核心工作流）。
- **命令特定帮助**：通过基于 switch 的分发函数，为单个命令（`init`、`try`、`sync` 等）提供详细的帮助信息。
- **保持描述更新**：维护最新的命令描述，包括实验性功能（如 `view` 命令运行时）。

## 重要不变性
- **无业务逻辑**：该模块不得导入或调用任何管道、持久化或核心领域逻辑。它严格属于表示层。
- **副作用受限**：唯一允许的副作用是控制台输出。此处不进行文件 I/O、配置更改或状态变更。

## 范围外（负面范围）
- 任何 CLI 命令的执行或业务逻辑。
- 管道编排、持久化或核心领域逻辑。
- 配置的读取或写入（委托给配置模块）。
- Git 钩子管理逻辑（委托给钩子模块）。

## 公开接口（导出的行为）
- `printGeneralHelp()` – 打印完整的命令列表及其类别。
- `printCommandHelp(cmd: string)` – 打印由名称标识的特定命令的详细帮助信息。

## 架构意图
该模块作为渲染帮助文本的薄 CLI 适配器，确保命令路由与业务逻辑清晰分离。它通过避免对内部领域模块的任何依赖，强制执行 **cli-入口点分离** 规则。
