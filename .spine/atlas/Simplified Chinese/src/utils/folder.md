# `src/utils` — ArchSpine 核心工具模块

本目录提供支撑 ArchSpine 镜像系统运行的基础设施与核心工具模块，按功能分为三个主要组别：

1. **代理指令管理** — 三个文件（`agent-instructions.sync.ts`、`agent-instructions.templates.ts`、`agent-instructions.ts`）共同负责仓库配置文件中代理指令块的同步、模板化及公共 API 暴露。其中 `.sync.ts` 执行文件级别的同步与移除操作，`.templates.ts` 存储静态指令块与忽略模式，`.ts` 作为统一导出入口。

2. **基础设施与文件操作** — `fs.ts` 提供原子写入与安全复制功能，`lock.ts` 实现基于文件的互斥锁保障运行时安全，`repo-path.ts` 对仓库路径进行跨平台归一化处理以确保数据库键值的一致性。

3. **命令行界面与展示** — `banner.ts` 渲染品牌横幅与版本信息，`confirm.ts` 处理交互式用户确认提示，`footprint.ts` 生成源码文件的确定性结构指纹。`git-hook.ts` 管理预提交 Git 钩子脚本块。

最关键的实现区域包括 `fs.ts` 中的原子文件操作、`lock.ts` 中的锁机制以及 `agent-instructions.sync.ts` 中的代理指令同步逻辑，这些模块直接影响数据完整性与系统可靠性。
