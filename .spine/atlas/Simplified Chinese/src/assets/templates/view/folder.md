该目录包含 ArchSpine 镜像系统的公开 API 文档与风险热点分析。两个核心文件共同描述了系统的外部接口与内部维护优先级。

- **public-surface.md** — 自动生成的文档，列出所有公开的 CLI 命令和标志、MCP 工具/资源/提示以及导出的模块符号。它是系统公开表面的唯一事实来源。
- **risk-hotspots.md** — 动态生成的风险报告，根据综合评分标识代码库中风险最高的文件，提供风险因素和影响的详细分析，帮助团队将维护工作集中在最关键的部位。

这两个文件使开发者能够清晰了解 ArchSpine 对外暴露了哪些接口，以及代码库内部哪些区域需要优先关注。其中，CLI、MCP 和模块入口点是最重要的实现区域，任何改动都会直接影响用户；风险热点数据则对确定重构与测试的优先级至关重要。
