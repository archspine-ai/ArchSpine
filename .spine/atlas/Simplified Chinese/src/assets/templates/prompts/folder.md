`prompts` 目录位于 `src/assets/templates` 下，是 ArchSpine 摘要生成流水线的提示模板引擎与模式定义中心。它包含了从项目和文件夹信息构建结构化架构摘要与图表所需的所有组件。该目录内的五个 TypeScript 模块各有明确职责：`schemas.ts` 提供定义语义摘要单元格式的 JSON 模式模板；`arch-diagram.ts` 包含用于生成结构化 JSON 架构图表的 LLM 提示模板；`blocks.ts` 提供纯工具函数，用于渲染结构化文本块，这些文本块嵌入到提示中；`examples.ts` 提供少样本示例，用于生成语义角色描述；`index.ts` 作为公共 API 外观，汇总并重新导出所有这些模块，便于外部调用。最重要的实现区域是架构图表模板与模式定义文件，它们直接决定了流水线的输出格式。开发者若想集成 ArchSpine 的提示引擎，应重点关注 `schemas.ts` 和 `arch-diagram.ts`，以理解结构约束与生成逻辑。
