此目录构成 ArchSpine 的提示工程工具包，包含用于构建大语言模型提示的所有可复用组件，以生成结构化的 JSON 格式摘要。其文件按功能分为以下四组：

- **架构图提示**（`arch-diagram.ts`）：根据项目和文件夹摘要生成完整的 LLM 提示，用于输出带有模式约束的 JSON 架构图规格说明。
- **块渲染工具**（`blocks.ts`）：提供纯函数，用于渲染身份块、指令块、上下文块、环境块、规则违反检查块、Git 意图块、JSON 模式块以及源内容块——这些是构建任何提示的核心基础组件。
- **示例数据**（`examples.ts`）：提供少量样本，定义角色描述的质量标准，用于引导模型生成正确的输出。
- **模式定义**（`schemas.ts`）：为 ArchSpine 的五种摘要单元（源文件、文档、配置文件、文件夹、项目）定义 JSON 模板，确保所有镜像任务中的输出结构一致。

`index.ts` 文件汇总并重新导出所有模式、渲染块和示例数据，为下游的提示生成模块提供单一入口点。这些组件共同构成一个模块化、可复用的层，驱动 ArchSpine 镜像系统中的语义摘要流水线。
