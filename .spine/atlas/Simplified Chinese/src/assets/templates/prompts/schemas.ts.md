# ArchSpine 架构定义模块

本模块提供 ArchSpine 镜像系统中所有语义摘要单元所使用的 JSON 模板结构。它定义了每个摘要必须遵循的规范形状，确保不同实体类型之间的一致性。

## 角色

为 ArchSpine 语义摘要单元提供 JSON 模板的架构定义模块。

## 主要职责

- 定义源文件语义摘要的 JSON 架构结构。
- 定义文档语义摘要的 JSON 架构结构。
- 定义配置文件语义摘要的 JSON 架构结构。
- 定义文件夹单元语义摘要的 JSON 架构结构。
- 定义项目级语义摘要的 JSON 架构结构。

## 重要不变性

- 所有导出的架构都是包含有效 JSON 模板字面量的字符串常量。
- 每个架构都将 `semantic` 对象定义为其根容器或主要容器。

## 排除范围（非职责）

- 本模块**不**对数据进行架构验证。
- 本模块**不**解析或解释被摘要文件的内容。
- 本模块**不**生成摘要；它仅提供模板结构。

## 公开接口（导出符号）

- `sourceFileSchema`
- `documentSchema`
- `configSchema`
- `folderSchema`
- `projectSchema`

这五个常量是唯一的外部可见行为。使用者导入相应的架构常量，并在为对应实体类型构建语义摘要时将其用作模板。
