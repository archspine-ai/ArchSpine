# ArchSpine – `sourceRoleExamples` 模块摘要

**角色：**  
该模块提供静态的、硬编码的少量示例，用于指导语义角色描述的生成。它是 ArchSpine 摘要流水线中提示工程的内容来源。

**主要职责：**
- 通过具体的正面和反面示例，定义角色描述的质量标准。
- 提供模板内容，用于塑造 AI 驱动的角色描述生成的提示。

**重要不变性与负面范围：**
- 导出的内容是一个只读的模板字面量，运行时永不修改。
- 该模块**不**执行动态示例生成、运行时逻辑或示例内容的验证/解析。

**导出的外部接口：**
- `sourceRoleExamples` – 唯一的公开符号，一个包含示例集的静态字符串常量。

**变更意图：**  
架构意图是提供规范、高质量的示例，确保系统的一致性和精确性。最近的变更恢复或更新了 LLM 编写的 Markdown 生成内容，很可能旨在改进用于提示的示例集。
