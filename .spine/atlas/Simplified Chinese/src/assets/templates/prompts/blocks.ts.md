# ArchSpine – 文本块渲染器

## 角色
纯工具函数，用于渲染结构化文本块，主要用于 ArchSpine 中的 LLM 提示构建。

## 主要职责
- 渲染身份块字符串，定义摘要任务的角色和目标。
- 从指令项数组渲染指令块字符串，并带有编号格式。
- 渲染上下文块字符串，包含标题和上下文数据，包裹在类似 XML 的标签中。

## 重要不变性与负面范围
- 函数是纯函数，无副作用。
- 输出是确定性的字符串拼接和格式化。
- 无外部依赖或运行时配置。
- **不**解析或解释渲染块的内容。
- **不**管理状态或副作用。
- **不**处理文件 I/O 或网络通信。

## 公开接口（导出的函数）
- `renderIdentityBlock`
- `renderInstructionsBlock`
- `renderContextBlock`
