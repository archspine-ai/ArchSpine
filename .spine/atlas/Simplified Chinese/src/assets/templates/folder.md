`src/assets/templates` 目录是 ArchSpine 镜像系统的核心模板资产库。它提供了可复用的基础组件，以支持整个项目的一致开发、维护与运行。该目录下包含四个具体子模块：

- **atlas**：定义镜像系统的架构文档，包括架构原则、同步机制、参数定义、维护边界及生命周期约束，确保各节点间的数据反射一致。
- **prompts**：实现提示工程工具包，包含用于通过大语言模型生成结构化语义摘要的模板、渲染块、示例和模式，覆盖源文件、文档、配置、文件夹和项目。
- **rules**：建立项目配置与代码质量标准，包括分层架构约束、导入规则和命名约定（如接口前缀、测试文件后缀），保证单体仓库中代码的一致性和关注点分离。
- **view**：提供公共 API 接口文档和风险热点分析，涵盖 CLI、MCP、模块及内部风险区域，帮助开发者理解外部接口并确定维护优先级。

关键实现领域包括：提示生成工具包（确保大语言模型输出的一致性）、架构同步规则（维持节点间数据一致性）以及公共 API 文档（指导外部集成）。这些模板共同为 ArchSpine 镜像系统的持续开发、可维护性和运行可靠性提供支撑。
