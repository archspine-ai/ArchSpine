# ArchSpine 使用报告工具

## 角色
ArchSpine CLI 工具函数，用于从清单数据库生成格式化的使用和审计报告。

## 主要职责
- 从清单中检索总使用统计、摘要分解和活跃违规。
- 格式化并输出人类可读的报告到运行时 IO 通道。
- 优雅处理空数据场景，显示适当消息。
- 计算并显示令牌数量和预估成本。

## 重要不变项与负面范围
- **依赖：** 清单接口用于数据检索，运行时 IO 接口用于输出格式化。
- **假设：** 清单数据库在提供的根目录下可访问。
- **不在范围内：** 清单数据库的数据持久化或修改；用户认证或授权；交互式命令行参数解析；图形用户界面（GUI）渲染。

## 最重要的导出行为
- **`runUsageReport`** — 执行报告生成和输出的主要公开函数。

## 架构意图
提供一个解耦、可复用的报告函数，供 CLI 和其他服务显示使用和审计数据。属于更广泛的 CLI 模块化重构和核心基础设施服务解耦的一部分。
