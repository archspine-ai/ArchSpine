# 修复服务公共 API 外观文件

本文件作为修复服务模块的**公共桶导出**文件。它不包含任何实现逻辑，而是从底层服务实现中重新导出关键符号，为消费者提供清晰、稳定的接口。

## 职责

- 重新导出 `FixService` 类和 `runFix` 函数。
- 重新导出 `FixRunSummary` 类型供外部消费者使用。
- 作为所有修复相关公共 API 表面的单一入口点。

## 不包含的内容

- **不**实现任何修复逻辑或业务规则。
- **不**编排服务执行流程。
- **不**提供命令行入口点。

## 不变约束

- 必须**仅**包含来自底层服务模块的重新导出。
- 必须**不**包含任何可执行逻辑或状态。

## 公共表面

- `FixService`
- `runFix`
- `FixRunSummary`

## 变更意图

架构意图是为修复服务提供一个稳定、解耦的公共 API，以促进模块化消费和依赖管理。此文件未检测到任何近期变更。
