# `extractRuleKeywords` — 关键词提取工具

## 角色
从架构规则文本中提取并过滤关键词的引擎工具函数。

## 主要职责
- 使用外部 `extractSearchTokens` 函数将规则内容分词为搜索令牌。
- 过滤掉预定义的规则相关停用词，以提取有意义的关键词。
- 将关键词列表限制为最多 16 个唯一令牌，以保证效率。

## 重要不变性与负面范围
- 不得依赖 CLI 入口点或服务编排模块。
- 必须是专注于文本分词和过滤的纯函数。
- **不**解析或验证规则结构。
- **不**对规则相关性进行评分或排序。
- **不**直接与 CLI 或服务编排交互。

## 最重要的导出行为
- **`extractRuleKeywords`** — 唯一的公开接口；调用者传入规则文本，返回一个去重、过滤停用词后的关键词列表，最多 16 个。
