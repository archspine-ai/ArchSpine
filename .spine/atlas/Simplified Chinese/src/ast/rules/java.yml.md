# ArchSpine 语法模式

## 目的
本文档定义了 ArchSpine 用于识别和分类 Java 代码元素（如导入、类、接口和方法调用）的模式规则。它是代码结构分析的语法层。

## 上下文与受众
适用于配置或扩展 ArchSpine 代码解析功能的开发者和 AI 代理。任何需要定义源代码结构如何被识别并映射到镜像系统中语义角色的人都会用到此文档。

## 主要职责
- 指定导入、类、接口和方法调用的模式
- 将代码结构映射到语义种类（Class、Interface）
- 提供代码使用检测的语法规则

## 不涉及范围
- 特定语言的运行时行为或执行语义
- 非 Java 语言支持或通用模式匹配

## 关键要点
- 定义了导入、类（公有和默认）、接口（公有和默认）以及方法调用的模式。
- 每个模式都映射到一个语义种类（例如 Class、Interface），用于结构化分析。
- 使用模式可检测方法调用和对象实例化。
