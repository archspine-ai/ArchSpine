# ArchSpine 符号提取模式

## 目的
本文档定义了 ArchSpine 镜像系统中用于解析 Python 源代码符号的模式提取规则。它描述了用于从源文件中识别导入、导出和使用的模式模板，从而实现自动化的依赖追踪和符号映射。

## 上下文与受众
适用于维护 ArchSpine 代码分析管道的开发者和 AI 代理。读者应了解 Python 语法以及基于模式的代码提取概念。本文档作为符号解析器组件的配置模式。

## 关键要点
- 三大模式类别：导入、导出和使用
- 导出涵盖函数、类、异步函数、变量和 `__all__` 列表
- 模式使用 `$$$SYMBOLS`、`$NAME`、`$VAL`、`$OBJ` 和 `$$$ARGS` 作为占位符
- 本文档是配置文档，而非运行时规范

## 职责
- 指定用于符号解析的导入模式（from-import 和简单导入）
- 定义函数、类、异步函数、变量和 `__all__` 列表的导出模式
- 描述调用表达式和属性访问的使用模式

## 不涉及范围
- Python 以外的语言特定语法
- 运行时行为或执行语义
- 项目特定的业务逻辑或领域规则
