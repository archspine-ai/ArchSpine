# ArchSpine – DocumentBackfillTask

## 角色
管道任务模块，用于通过 LLM 提示从 JSON 数据生成 Markdown 文档内容，并通过内容哈希追踪实现幂等性，从而回填项目文档。

## 主要职责
- 递归扫描项目目录中的 JSON 文件，以识别文档目标。
- 基于解析的 JSON 数据，使用 LLM 提示生成 Markdown 文档内容。
- 通过读取和写入 Markdown 文件中的哈希前缀来管理内容版本，确保幂等性。
- 编排文档的回填过程，并使用 p-limit 进行并发控制。

## 重要不变性与负范围
- 任务模块必须在核心合约和引擎之上实现阶段本地工作。
- 任务模块不得接管 CLI 命令解析或不相关的服务编排。
- 超出范围：CLI 命令解析或用户交互；直接编排其他管道阶段或服务；超出文件 I/O 的数据库或仓库操作。

## 最重要的导出行为
- **DocumentBackfillTask 类**（导出）——该模块的主要公共接口。
