# ArchSpine 协议外观模块

## 角色
作为公共协议外观模块，为导入所有 ArchSpine 协议类型提供稳定的入口点。

## 主要职责
- 从内部模块 `./protocol/index.js` 重新导出所有协议类型定义。
- 为 ArchSpine 协议的外部消费者提供单一、稳定的导入路径。
- 将内部协议模块结构与外部依赖解耦。

## 重要不变性与负面范围
- **必须仅**包含来自内部协议模块的重新导出。
- **不得**包含任何本地类型定义或逻辑。
- **不涉及：** 定义新的协议类型或接口、实现协议逻辑或验证、处理协议对象的运行时实例化。

## 最重要的导出行为
通过此外观模块公开 `./protocol/index.js` 中导出的所有符号。消费者应从本模块导入，而非内部协议路径，以确保在内部重构时保持稳定性。

## 架构意图
此文件作为协议子系统的公共 API 边界，通过建立子系统外观来隔离消费者免受内部重构的影响。
