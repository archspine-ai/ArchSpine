# ArchSpine 变更日志摘要

## 目的
本文档是 ArchSpine 的官方变更日志，记录每个公开发布版本中的所有重要变化。它旨在帮助开发者、贡献者和用户快速了解版本之间的新增、改进或修复内容。

## 背景与受众
主要受众包括使用或贡献 ArchSpine 的开发者、系统集成人员以及任何关注项目演进的人。变更日志仅涵盖公开发布版本，从 1.0.0 开始，不包括内部预发布历史。

## 关键要点
- 首个公开发布版本（1.0.0）引入了核心工作流：init、sync、check、fix 和 MCP 服务器。
- 新增功能包括 Git 集成的产物策略选择和失败重试同步。
- 安全性和可靠性改进包括原子写入、运行时锁和严格的 schema 版本强制。
- 文档现在包括双语 README 和文档站点入口。
- 重构包括模块化的 CLI init 流程和托管文件的对称回滚。
- 旧版 schema（0.4.0）不再受支持，需要重建。

## 范围与边界
- **涵盖范围**：跟踪每个公开发布版本中的新功能、安全增强、文档更新、重构和测试变更。维护从 1.0.0 开始的版本化变更记录。为开发者和用户提供清晰、结构化的历史记录。
- **不涵盖范围**：首个公开发布之前的内部开发历史。详细的错误报告或问题跟踪。超出当前版本范围的未来路线图或计划功能。
