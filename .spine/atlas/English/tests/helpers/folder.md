This directory contains `lock-worker.mjs`, a utility script that provides a test harness for exercising the LockManager’s acquire and release behavior. The script demonstrates lock acquisition and release across multiple modes, outputs status messages as JSON for external monitoring or integration, and handles OS signals (e.g., SIGINT, SIGTERM) to enable graceful shutdown. Its core implementation areas include lock-mode enumeration (shared, exclusive), timeout handling, signal-driven cleanup, and structured reporting. As the sole module in this directory, it serves as both a development debugging tool and a reference for correct LockManager usage.
