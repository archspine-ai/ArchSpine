# `scripts/` — Build, Test, Release & Demo Automation

This directory contains all automation scripts that support the ArchSpine project's lifecycle, from development and testing through to release and demonstration. The scripts are grouped by purpose:

- **Build & Development** — `build.mjs` orchestrates compilation, asset copying, and cleanup for distribution; `dev-build.mjs` provides a watch-mode development build.
- **Verification & Benchmarking** — `final-verify.mjs` runs end-to-end verification of ASTExtractor multi-language export extraction; `phase5-live-bench.mjs` benchmarks the AST-based semantic extraction pipeline.
- **Release & Publishing** — `release-gate.mjs` enforces release readiness checks; `publish-placeholder.mjs` handles placeholder package publishing.
- **Database** — `update-db-schema.cjs` manages schema migrations for the ArchSpine cache system.
- **Demonstration** — `demo.tape` and `project-demo.tape` are terminal recording scripts that showcase the CLI workflow and generate a visual preview.

Key implementation areas: build orchestration, end-to-end verification, live benchmarking, release gating, and database schema migration.
