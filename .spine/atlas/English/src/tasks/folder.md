# `src/tasks` — Pipeline Task Implementations

The `src/tasks` directory is the operational heart of the ArchSpine mirror system. Every file here implements a discrete stage of the core processing pipeline, from initial file scanning through to database commitment and view derivation. These tasks are designed to be composed into a sequential or partially parallel workflow that transforms raw source code into a richly documented, architecturally validated, and indexed mirror.

## Notable Children and Groupings

The tasks can be grouped into several logical phases:

- **Scanning & Reconciliation:** `scan-cleanup.ts` and `reconcile.ts` handle file system discovery, orphan cleanup, and manifest synchronization.
- **AST Extraction & Summarization:** `ast-extra.ts` extracts AST symbols, while `lite-summarize.ts` and `summarize.ts` generate lightweight and full LLM-based semantic summaries respectively.
- **Documentation & Validation:** `document-backfill.ts` generates markdown documentation from JSON via LLM prompts, and `validate.ts` checks architectural rule compliance.
- **Fixing & Indexing:** `fix.ts` applies automated LLM-driven fixes for violations, and `reverse-index.ts` builds reverse dependency edges.
- **Commit & Derivation:** `state-commit.ts` persists file state to SQLite, `aggregate.ts` performs hierarchical content aggregation, and `post-commit-derivation.ts` orchestrates view, aggregation, and reverse index derivation. `view-derivation.ts` specifically derives public surface, risk hotspots, and architecture diagrams.

## Key Implementation Areas

The most critical implementation areas are:

- **LLM Integration:** `summarize.ts`, `lite-summarize.ts`, `document-backfill.ts`, `validate.ts`, and `fix.ts` all rely on LLM calls for semantic understanding, documentation generation, validation, and automated repair.
- **State Management:** `state-commit.ts` and `reconcile.ts` ensure data integrity between the file system, manifest, and SQLite database.
- **View Derivation:** `view-derivation.ts` and `post-commit-derivation.ts` produce the high-level architectural views that make the mirror system useful for human and AI consumption.
