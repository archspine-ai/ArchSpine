The `src/infra/llm/providers` directory contains the concrete LLM provider implementations and supporting utilities that enable ArchSpine's mirror system to generate semantic summaries. It is organized into three provider clients and one shared utility module.  

The provider implementations are `gemini.ts`, which wraps Google's Gemini API for structured summary generation; `openai.ts`, an OpenAI-compatible client that also orchestrates prompt assembly for the semantic analysis pipeline; and `mock.ts`, a test dummy that simulates LLM responses for unit and integration testing.  

The `utils.ts` module provides shared infrastructure: it assembles content into prompts, parses structured responses, and merges usage data across providers. Together, these files form the backbone of ArchSpine's semantic summary generation for various file kinds.
