# ArchSpine Prompt Infrastructure (`src/infra/prompt`)

This directory houses the prompt construction and assembly layer of the ArchSpine system. Its core responsibility is to provide a comprehensive toolkit for building structured, localized prompts that drive AI agent interactions. The infrastructure is organized around several key modules:

- **`aggregate.ts`** – A factory for semantic synthesis, producing structured prompt strings for documents, configuration files, directories, and project units via a `PromptBuilder` pipeline.
- **`builder.ts`** – The central orchestration class that assembles structured prompt blocks for AI agent conversations.
- **`markdown.ts`** – A utility for constructing localized, markdown-only prompt strings from semantic JSON input.
- **`shared.ts`** – Generates formatted output contract strings that define the expected response format for the prompt system.
- **`source.ts`** – A facade function that creates structured LLM prompts specifically for source code analysis tasks.
- **`types.ts`** – Defines the TypeScript type system for the prompt generation and response formatting protocol.

The most critical implementation areas are the `PromptBuilder` pipeline in `aggregate.ts` and the output contract generation in `shared.ts`, as they directly control how prompts are structured and how responses are validated.
