The `src/infra/prompt` directory houses the prompt construction and assembly infrastructure for the ArchSpine system. It provides a full toolkit for building structured, localized prompts that AI agents can consume. The children are grouped around core assembly (e.g., `aggregate.ts` as a factory for semantic synthesis, `builder.ts` as a utility that orchestrates the prompt pipeline), output contract generation (`shared.ts` for formatted response strings), specialized prompt constructors (`markdown.ts` for localized markdown-only prompts, `source.ts` for source code analysis prompts), and type definitions (`types.ts` for the protocol). Key implementation areas include the prompt construction pipeline, multi-language localization support, and output formatting for AI agent responses.
