The `infra` directory is the metadata and indexing layer for the database infrastructure within the ArchSpine mirror system's index. It contains two configuration files that together provide structural metadata, content integrity tracking, and provenance records for database-related source code.

- **`database.ts.json`** – Holds the metadata and structural index for a TypeScript database infrastructure source file. This file records the file's role, dependencies, and any extracted semantic information.
- **`folder.json`** – Defines the structural metadata and indexing provenance for the entire source directory itself. It captures the directory's role in the mirror system and the pipeline stages used to generate the index (AST and LLM).

These two files form the core of the database infrastructure indexing, enabling both human and AI consumers to understand the organization, dependencies, and processing history of the database layer.
