ArchSpine

A rule-enforced mirror system for documenting, analyzing, and synchronizing software architecture across distributed nodes.

Core Modules

  • CLI, MCP Server, and Services orchestrate user and AI agent interactions.
  • Engines, AST Parser, and Tasks drive scanning, analysis, and validation pipelines.
  • View Generator produces architectural diagrams and reports.

Key Dependencies

  • SQLite for persistent state and indexing.
  • LLM Client for AI-powered summarization and analysis.
  • Config Manager, Credential Store, and Manifest Manager for infrastructure.

System Boundaries

  • External LLM providers for semantic analysis.
  • File system for source code scanning and output generation.
  • Test suite validates all core components and pipelines.
